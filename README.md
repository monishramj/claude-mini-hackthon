# The Pocket Pundit 🏀
###### This project was built for the Claude Mini Hackathon (3/25) - Sports Branch. Teams were given 20 minutes to create a simple project and pitch to present. 
 
🔗 check it out here!! https://monishramj.github.io/claude-mini-hackthon/

a lightweight, vanilla mock web application built to generate March Madness basketball analysis to lower the difficultly for entry into basketball statistics and March Madness.
provides users with randomized "expert takes" and translates them into plain English for all to understand.
